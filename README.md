# 📺 YouTube Channel Landing Page

This repository contains the complete source code for the **YouTube Channel Landing Page** created in my YouTube tutorial.

If you're watching the tutorial and want to compare your code, find mistakes, or download the completed project, you're in the right place.

## 📂 Project Overview

This is a **static webpage** built using **HTML5** and **CSS3**. The project is designed to help beginners understand how to structure and style a modern landing page.

### The webpage includes:

* 🧭 Navigation Bar
* 🏠 Home Section
* 👤 About Section
* 🎥 Latest Videos Section
* 📞 Contact Section
* 📌 Footer

## 📁 Project Assets

All the images used in this project are available in the **`images`** folder. Feel free to download and use them while following the tutorial.
